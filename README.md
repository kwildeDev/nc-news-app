## Project Overview

NC News is a social news aggregation website built to demonstrate full-stack development skills. Key features include article voting, comment posting, and topic-based content filtering.

### Frontend: Mobile-First Experience

- Technology: Built with React and Material UI.

- Design: Features a mobile-first approach and responsive design for a seamless user experience.

- Architecture: Utilizes reusable components and effective state management.

### Backend: Robust API and Data Handling

- Technology: Powered by Node.js and Express.

- API: Implements a RESTful API with CRUD operations for articles, voting, and comments.

- Testing & Data: Features 56 Jest tests for robust validation and uses PostgreSQL for data storage.
